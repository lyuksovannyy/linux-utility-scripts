# linux-utility-scripts

`push-to-talk.py` - ability to globally bind an key to mute/unmute access to microphone for an app(s)
- by default on f13 mutes **all apps** beside **vesktop** and **gpu screen recorder**
- on f15 mutes **vesktop** and leaves all **other apps unmuted**
- and makes sure to **unmute all apps** if both keys are pressed
